# ExpenseTracker

It is an [Expo Managed] Project

## Step 1
Install all the dependencies using below commands: 
```bash
# Using npm
$ npm install

# or using yarn
$ yarn install
```

## Step 2
Run the project using below command
```bash
$ npm expo start

# or to run on android
$ npm run android

# or to run on IOS
$ npm run ios
```
